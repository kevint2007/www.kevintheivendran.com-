#### The "Inconvenience Test" for Passion
*December 2025*

If you don't mind working on something when it is inconvenient to do so, that means you are truly passionate about it.

I realized this with school assignments. Some projects felt like chores because I wasn't interested in the topic. However, with assignments that I found interesting, I would spend extra time going down "rabbit holes," learning about the topic far beyond what was required.

I think applying this "inconvenience test" is a useful framework for determining what you should actually pursue in life.
